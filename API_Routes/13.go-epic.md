# Error Handling Practice Routes

```http

POST /reviews
→ Return 400 if fields missing

PATCH /bookings/:bookingId
→ Return validation error

DELETE /conflicts/:conflictId
→ Return proper delete response

GET /admin/reviews
→ Return 401 if unauthorized

POST /auth/login
→ Return invalid credentials

GET /users/:userId
→ Return user not found

PATCH /users/:userId
→ Return invalid update data
```
